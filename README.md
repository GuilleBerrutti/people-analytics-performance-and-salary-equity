# 📊 People Analytics - Dashboard de Gestión de Talento

## 📝 Descripción del Proyecto
Este proyecto simula un análisis real de Recursos Humanos para una empresa de 50 empleados. 
El objetivo es proporcionar una herramienta visual interactiva que permita a la gerencia tomar
decisiones basadas en datossobre la retención de personal y la distribución presupuestaria.

## 🛠️ Tecnologías Utilizadas
* **Microsoft Excel:** Limpieza de datos (ETL), normalización de campos y cálculos de antigüedad mediante funciones de fecha.
* **Power BI:** Modelado de datos, creación de columnas condicionales y diseño de interfaz de usuario (UI) en Dark Mode.

## 💡 Análisis e Insights Clave
* **Alerta de Rotación:** El departamento de **Ventas** presenta una tasa de rotación del **20%**, la más alta de la organización, lo que sugiere una revisión urgente del clima laboral o incentivos.
* **Liderazgo Salarial:** El área de **IT** tiene el promedio salarial más alto ($3.000), alineado con las tendencias del mercado tecnológico.
* **Salud Organizacional:** La tasa de retención global es del **86%**, un indicador sólido para una empresa en crecimiento.
